# Youtube Style ID

> Generating youtube style id


## Getting Started

Install the module with: `npm install youtubeid`

```js
var youtubeid = require('youtubeid');
```

## Examples

youtubeid.generate();
options
-chars = number of characters in the id

var chars = 5
youtubeid.generate(chars);
//=> Wc8xF
